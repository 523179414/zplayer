# zplayer

#### 项目介绍
史上最精简的音乐播放器！300多行js！你想要的功能zplayer都有！请加Q群130512958交流技术。zplayer还支持初始化多个播放器。如果使用有建议或者其他问题，可以加群交流。

#### 使用说明
1. down下来本项目后，在页面内引入 css/zplayer.min.css和js/zplayer.min.js
